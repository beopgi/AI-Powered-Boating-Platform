# 🌊 AI-Powered Boating Life Platform

> **데이터 기반 해양 라이프스타일 플랫폼**
>
> 항해 데이터, 위치 정보, 공공데이터, AI 추천을 결합하여 사용자의 보팅 라이프를 더 스마트하고 안전하게 만드는 올인원 모바일 서비스입니다.

---

## 📌 Project Overview

**AI-Powered Boating Life Platform**은 해양 액티비티를 즐기는 사용자들을 위해 설계된 **Product-Ready 수준의 Flutter 모바일 애플리케이션**입니다.  
단순한 기록 앱을 넘어, 실제 항해 흐름에서 발생하는 데이터를 기반으로 사용자 맞춤형 서비스를 제공합니다.

### Why this project?
해양 라이프스타일 서비스는 데이터 파편화, 예약/탐색 경험의 단절, 기록 관리의 번거로움 같은 문제를 안고 있습니다. 본 프로젝트는 아래 문제를 해결합니다.

- 항해 기록이 흩어져 관리가 어려운 문제
- 해양 활동 추천 정보가 개인화되지 않는 문제
- 지도 기반 탐색과 이벤트 예약이 분리된 사용자 경험

### Core Value
- 🧠 **AI 개인화 추천**으로 사용자 성향에 맞는 해양 활동 제안
- 📍 **위치·지도 기반 탐색**으로 즉시성 높은 탐험 경험 제공
- 🗂️ **자동 항해 로그 관리**로 기록의 신뢰성과 활용도 향상
- 🎟️ **유람선 이벤트 예약 통합**으로 서비스 전환 비용 최소화

---

## ✨ Key Features

- 🧠 **AI 기반 해양 라이프스타일 추천**  
  항해 이력, 위치, 선호 패턴을 분석하여 맞춤형 액티비티/코스를 추천합니다.

- 🛥️ **항해 기록 자동 저장**  
  주행 경로, 이동 시간, 주요 포인트를 자동으로 기록하고 히스토리로 관리합니다.

- 🗺️ **지도 기반 탐색 기능**  
  주변 해양 포인트, 추천 코스, 주요 시설을 지도에서 직관적으로 확인할 수 있습니다.

- 🎫 **유람선 이벤트 실시간 예약**  
  이벤트 조회부터 예약까지 앱 내에서 빠르게 처리할 수 있습니다.

- 🛍️ **해양 라이프스타일 제품 서비스**  
  해양 활동 관련 상품/서비스를 탐색하고 이용할 수 있는 확장형 구조를 제공합니다.

- 🌐 **다국어 지원 (한국어 / 영어)**  
  국내외 사용자 대응을 위한 국제화(i18n) 기반을 포함합니다.
